# npm-publish-mcp

MCP 服务器，用于 npm 登录/发布/查看包操作。

## 工具

| 工具 | 功能 |
|------|------|
| `npm-token-status` | 检查 NPM_TOKEN 状态和登录状态 |
| `npm-login` | 登录指引：获取和设置 NPM_TOKEN |
| `npm-publish` | 发布包到 npm registry（自动预检） |
| `npm-view` | 查看 registry 包信息 |
| `npm-logout` | 退出登录指引 |

## 使用

设置环境变量 `NPM_TOKEN`：

```bash
export NPM_TOKEN="npm_xxxxxx..."
```
